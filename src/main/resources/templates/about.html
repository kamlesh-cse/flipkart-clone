About Us

Who We Are

FlipkartClone is a college mini-project that simulates an e-commerce platform similar to Flipkart. Built using Spring Boot, Thymeleaf, and MySQL, this project demonstrates core web development concepts including MVC architecture, authentication, and database management.

Our Mission

To provide a simple yet functional e-commerce experience that helps students understand how real-world online shopping platforms work behind the scenes.

Technologies Used

Java

Spring Boot

MySQL

Project Team

This project was developed as part of our college mini-project for the Web Development course. It showcases the practical implementation of concepts learned during the semester.
